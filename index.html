What is Microsoft PowerToys?

PowerToys is a free toolkit for Windows power users by Microsoft. It lets you customize, control, and boost productivity on Windows with lots of helpful mini-tools.

  • Pin any window on top (like calculators or videos)
  • Quickly find your mouse cursor
  • Extract text from anywhere on screen (OCR)
  • Mark or highlight parts of your screen
  • And much more!

How to Install PowerToys

  1. Go to the PowerToys Releases page or Microsoft Store.
  2. Click Download for your Windows version and open the installer.
  3. Or, open Command Prompt and run: winget install Microsoft.PowerToys
  4. After install, launch PowerToys from Start menu.
